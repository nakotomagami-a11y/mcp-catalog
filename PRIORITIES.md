# Research Priorities

When scanning for new MCP servers, prioritize in this order:

## High Priority
- **Open-source servers** (any category)
- **Developer Tools** — git, CI/CD, linting, testing, code analysis
- **Security** — SAST, DAST, vulnerability scanning, pentesting, secrets management
- **AI/ML** — model serving, RAG, embeddings, agent frameworks
- **Marketing** — CMS, email, SEO, social media, analytics

## High-Medium Priority
- **Finance & Trading** — market data APIs, crypto, portfolio, payments
- **Databases** — any new DB adapters

## Medium Priority
- **Cloud & Infra** — AWS, GCP, Azure, Docker, K8s
- **Productivity** — project management, calendar, notes

## Lower Priority (still include)
- Creative, Game Engines, Communication, Commerce, Science, Education
